We show that for any integer $n$, $n^2 \bmod 4$ is either $0$ or $1$.
Consider two cases based on the parity of $n$.
If $n$ is even, write $n = 2k$. Then $n^2 = 4k^2$, which is $0 \bmod 4$.
If $n$ is odd, write $n = 2k+1$. Then $n^2 = 4k^2 + 4k + 1$, which is $1 \bmod 4$.
In both cases $n^2 \bmod 4$ is $0$ or $1$, as claimed.
